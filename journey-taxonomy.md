## 用户旅程（视图模式）

### 按工作流阶段

- **矩阵结构：** 左侧 L1/L2 工作流任务为行，四开发者角色为列
- **是否选角色：** 否（四角色同时横向对照）
- **任务范围：** 各角色已配置任务白名单的并集（当前 17 项）

### 按开发者角色

- **矩阵结构：** 选定角色后，左侧 L1/L2 为行，单列展示该角色配对提问
- **是否选角色：** 是
- **任务范围：** 该角色任务白名单；未配置白名单的角色显示全部 26 项

## L1 / L2 工作流阶段（全量 26 任务）

### L1 · 环境与安装（3）

- **I** 版本兼容排查
- **J** 安装与环境变量
- **K** 容器 / 镜像搭建

### L1 · 算子开发（6）

- **D** 自定义算子开发
- **L** 算子精度排查
- **M** 动态 shape / Tiling
- **N** 算子融合
- **O** 注册与框架集成
- **C** 算子库选型

### L1 · 训练（4）

- **F** 分布式训练配置
- **P** 混合精度训练
- **Q** 显存 / OOM 优化
- **R** 精度 / 收敛排查

### L1 · 推理与部署（4）

- **A** 模型转换 / 导出
- **S** 推理服务部署
- **H** 量化
- **T** 动态 batch / shape 推理

### L1 · 性能优化（4）

- **B** Profiling 定位瓶颈
- **U** 访存 / occupancy 优化
- **V** 计算与传输重叠
- **W** 多卡通信优化

### L1 · 调试（2）

- **X** 内存越界定位
- **E** 报错码 / 异常排查

### L1 · 迁移（3）

- **G** CUDA → 昇腾 迁移
- **Y** 跨芯片迁移
- **Z** 概念心智模型对照
